# Existing Content Deck

## Goal

- Generate a PPTX from existing notes.
- Keep the content editable.
- Use images only when they are the right object type.

## Table Example

| Path | Main strength | Editability |
| --- | --- | --- |
| Native PPTX | Maintainable | High |
| Image-first deck | Fast visual draft | Low |
| Mixed deck | Balanced | Medium |

## Closing
